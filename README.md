# 🍽️ BiteBrowser – Recipe Search App

## 📌 Overview
BiteBrowser is a web app that lets users search for recipes and view ingredients and instructions using a public API.

---

## 🚀 Features
- 🔍 Search recipes by name  
- 📦 Fetch data from API  
- 🧾 View ingredients & instructions  
- 🔼 Sort recipes by ID  
- 🎯 Filter by category  
- 🪟 Popup modal for recipe details  
- ⚡ Loading & error handling  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- TheMealDB API
