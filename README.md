# 🍽️ BiteBrowser – Recipe Search App

## 🌐 Live Demo

https://arivgupta07-byte.github.io/BiteBrowser/

---

## 📌 Overview

BiteBrowser is a web application that allows users to search for recipes and view ingredients and cooking instructions using a public API.
It provides a clean and interactive interface with features like sorting, filtering, and detailed recipe popups.

---

## 🚀 Features

* 🔍 Search recipes by name
* 📦 Fetch data from API
* 🧾 View ingredients & instructions
* 🔼 Sort recipes by ID
* 🎯 Filter by category
* 🪟 Popup modal for recipe details
* ⚡ Loading & error handling

---

## 🧠 Concepts Used

* Fetch API
* DOM Manipulation
* Higher Order Functions (`filter`, `sort`, `forEach`)
* Event Handling

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* TheMealDB API
